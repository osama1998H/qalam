# قلم - Qalam

محرر أكواد عربي مصمم خصيصاً للغة ترقيم، مبني باستخدام Rust.

Arabic RTL Code Editor for Tarqeem, built with Rust.

<div dir="rtl">

## نظرة عامة

قلم هو محرر أكواد عربي من اليمين إلى اليسار (RTL)، مصمم لحل مشاكل كتابة الأكواد العربية في المحررات التقليدية مثل VSCode.

### المميزات

- ✅ دعم كامل للنصوص العربية من اليمين إلى اليسار
- ✅ تشكيل الحروف العربية بشكل صحيح
- ✅ تلوين أكواد ترقيم
- ✅ لوحة ملفات عربية
- ✅ تكامل مع خادم لغة ترقيم (LSP)
- ✅ سمات داكنة وفاتحة

## التثبيت

### المتطلبات

- Rust 1.70+
- مترجم ترقيم (للتكامل مع LSP)

### البناء من المصدر

</div>

```bash
# استنساخ المشروع
git clone https://github.com/osamamuhammed/qalam.git
cd qalam

# البناء
cargo build --release

# التشغيل
cargo run --release
```

### اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|---------|---------|
| `Cmd+S` / `Ctrl+S` | حفظ الملف |
| `Cmd+O` / `Ctrl+O` | فتح ملف |
| `Cmd+N` / `Ctrl+N` | ملف جديد |

## البنية

</div>

```
qalam/
├── crates/
│   ├── qalam/          # التطبيق الرئيسي
│   ├── qalam-core/     # المحرك الأساسي
│   ├── qalam-text/     # معالجة النصوص العربية
│   ├── qalam-ui/       # واجهة المستخدم
│   ├── qalam-syntax/   # تلوين الأكواد
│   └── qalam-lsp/      # عميل LSP
└── assets/
    └── fonts/          # الخطوط العربية
```

## المكتبات المستخدمة

- **iced** - إطار واجهة المستخدم
- **cosmic-text** - معالجة النصوص مع دعم RTL
- **ropey** - بنية بيانات النصوص
- **unicode-bidi** - خوارزمية الاتجاه ثنائي اللغة
